# 选品雷达

家居用品与清洁用品的公开信息趋势榜单。包含排行榜页面、轻量接口与每日采集脚本。

分享给其他人使用、上传 GitHub 和部署公网时，请查看当前目录中的 `DEPLOY.md`。

当前榜单按照“淘宝式一级类目 -> 二级类目 -> 单品”拆分。每个二级类目显示 TOP100。

已包含的大类：

- 居家日用
- 食品饮料
- 消毒杀菌用品
- 母婴用品
- 美妆个护
- 住宅家具
- 床上用品

例如“居家日用 -> 拖把/拖布”会展示 TOP100 单品。

## 启动

在项目根目录运行：

```bash
npm start
```

打开 `http://127.0.0.1:4173/radar/`。主页导航中的“家居雷达”也会进入这个页面。

仅查看页面时，也可以直接双击 `open-offline.command` 或 `index.html`。离线预览支持榜单、筛选、搜索和商品详情；启动服务后可使用“立即更新”按钮。

需要实时更新时，也可以双击项目根目录的 `start-radar.command`，或当前目录的 `start.command`。它们都会启动统一 Web 服务并自动打开榜单页面。

## 每日更新

```bash
cd home-trend-radar
npm run crawl
```

服务器上可使用 cron 每天运行一次：

```cron
0 7 * * * cd /path/to/home-trend-radar && /usr/bin/npm run crawl
```

编辑当前目录中的 `data/sources.json`，可以添加有权访问的公开页面或授权接口：

- `jsonld`：读取公开页面中的 Schema.org `Product` JSON-LD。
- `json`：读取返回 `{ "products": [...] }` 的授权接口。
- `seed`：本地演示数据。

页面内置京东、淘宝、天猫、苏宁、抖音与小红书的公开检索入口。点击商品详情中的“打开商品来源”或来源状态中的“打开来源”，会跳转到对应公开页面。

- 京东、淘宝、天猫、苏宁：商品搜索依据。
- 抖音、小红书：内容搜索依据，部分页面可能要求登录。

公开检索入口便于人工核验，不等同于平台授权热榜接口；接入真实销量、点赞量、收藏量或日增长率时，应替换为已获授权的数据源。

采集器不会处理登录、验证码、绕过限制或受保护数据。接入平台前请确认服务条款、robots.txt 与适用的数据合规要求。
